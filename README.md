# AMOA-SEQ
AMOA-SEQ utilizes non-redundant AMOA sequences from NCBI as well as curated archaeal, bacterial and comammox AMOA sequences from Elves et al. 2018, Lee et al. 2023 (this study), Palomo et al 2022 for taxonomic and phylogenetic analyses, using an expert-guided approach to analyze diverse AMOA sequences.
